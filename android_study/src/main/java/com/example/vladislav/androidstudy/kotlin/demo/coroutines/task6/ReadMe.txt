Генерация числовой последовательности
Напиши программу на Kotlin с использованием корутин, которая генерирует числовую последовательность согласно следующему условию:

Последовательность начинается с двух чисел:
a1 = 1, а2 = 1

Формула выглядит так:
    a(n+1) = a(n) + k*n
k — постоянный коэффициент (целое положительное число).
n - порядковый номер

Пример:
Ввод:k = 2N = 5

Вывод:1, 1, 3, 7, 13
